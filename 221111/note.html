git hub 와 vscode 연동 (cloud처럼 사용하기)
1. vscode에서 새로운 폴더 불러오기
2. vscode 소스 초기화 후 터미널에 https://github.com/vincent94hong/WEB_workspace.git (깃허브 repositories 주소) 입력
3. 터미널에 git pull origin master 입력
4. 이후 불러온 코드 작업 후 소스에 change + 추가한 후 커밋
5. 커밋한 파일을 깃허브에서 보려면 터미널에 git push -u origin master 입력

*vscode 로그인된 깃허브 아이디 이외의 아이디에서 불러오거나 초기에 깃허브와 vscode가 링크 안될시
PS C:\Web_Workspace1> git config --global user.name vincent94hong (github ID)
PS C:\Web_Workspace1> git config --global user.email vincent94hong@gmail.com
git config 부터 터미널에 입력

tag a
1. 속성 href="하이퍼링크" 를 통해 해당페이지로 이동가능
2. 속성 target="blank" 를 통해 해당페이지를 새로운 탭에서 열기
3. 속성 title="HTML5 (enter) specification" 를 통해 하이퍼링크에 마우스를 댔을때, 박스형태로 정보를 제공


freenom
1. freenom 사이트에서 원하는 도메인을 작성 후 중복확인
2. 구글 아이디를 통해 도메인 받음


git hub ~ netlify
1. netlify 에 깃허브로 로그인을 한뒤, 깃허브의 repositories (main~master) 선택
2. netlify site-deploys-branchs 에서 repositories 확인
3. netlify domain settings-domains-add domain alias-freenom에서 받은 도메인 입력
4. 연동 완료
